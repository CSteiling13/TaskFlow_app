# TaskFlow – Full Stack Task Manager

A clean full-stack task management application built with React, Express, and SQLite. Users can create, manage, and track tasks with due dates and priorities.

## Features
- Create, update, and delete tasks
- Set due dates and priority levels
- Mark tasks as complete/incomplete
- Persistent storage using SQLite
- Clean and responsive UI

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: SQLite

## How to Run

### Backend
cd server
node server.js

Runs on:
http://localhost:3001

### Frontend
cd client
npm install
npm run dev

Runs on:
http://localhost:5173

## API Endpoints

GET /api/tasks  
POST /api/tasks  
PUT /api/tasks/:id  
DELETE /api/tasks/:id
